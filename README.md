# OrganizeNotes AI

AI-Powered Smart Study & Note Management Platform

## Features

- User Authentication (JWT)
- AI-powered Notes Summary
- AI-generated MCQs
- AI Interview Questions
- Smart Calendar
- Email Reminder System
- Pin Notes
- Task Completion
- Responsive Design
- Dark / Light Mode

## Tech Stack

Frontend
- React.js
- Tailwind CSS
- Axios

Backend
- Node.js
- Express.js

Database
- MySQL

Authentication
- JWT

AI
- Google Gemini API

Email
- Nodemailer

Deployment
- Frontend → Vercel
- Backend → Render
- Database → Railway
